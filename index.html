Clinical Question

Which type of hiatus hernia is this?

CT Findings

Herniation of stomach above diaphragm
Widened esophageal hiatus

Multiplanar CT

Radiological Interpretation

Gastric fundus above diaphragm
GE junction displacement
Hiatus widening >15 mm

Diagnosis

Type III (Mixed Hiatal Hernia)

Definition

Herniation of abdominal contents through esophageal hiatus into thorax

Epidemiology

< 40 years: ~10%
> 50–70 years: 30–50%
Slight female predominance

Clinical Presentation

Asymptomatic or:
Epigastric pain, fullness, nausea, vomiting
Poor correlation with GERD

Types

Type 1: Sliding
Type 2: Paraesophageal
Type 3: Mixed
Type 4: With other viscera

CT Features

Stomach above diaphragm
Hiatus widening (>15 mm)

Treatment

Symptomatic → Surgery
Asymptomatic → Conservative

Complications

Gastric volvulus
Ischemia
Respiratory failure

Differential Diagnosis

Retrocardiac abscess
Empyema
Esophageal diverticulum

Conclusion

CT is essential to identify structural abnormalities and prevent complications
