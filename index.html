Definition

Herniation of abdominal contents through esophageal hiatus into thoracic cavity

Epidemiology

  • < 40 years: ~10%
  • > 50–70 years: 30–50%
  • Female predominance

Types

Type 1 – Sliding
Type 2 – Paraesophageal
Type 3 – Mixed
Type 4 – Complex

CT Features

  • Stomach above diaphragm
  • Hiatus widening >15 mm
  • Crura separation

Treatment

Symptomatic → Surgery
Asymptomatic → Conservative

Complications

  • Gastric volvulus
  • Obstruction
  • Ischemia
  • Respiratory failure

Differential Diagnosis

  • Retrocardiac abscess
  • Empyema
  • Esophageal diverticulum
