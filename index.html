Quiz

Which type of hiatus hernia?

Definition

Hiatus hernia occurs when abdominal contents herniate through the esophageal hiatus into the thoracic cavity

Epidemiology

  • < 40 years → ~10%
  • > 50–70 years → 30–50%
  • Slight female predominance

Types

Type 1: Sliding (~95%)
Type 2: Paraesophageal
Type 3: Mixed
Type 4: Complex (with other organs)

CT Features

  • Stomach above diaphragm
  • Widened hiatus (>15 mm)
  • Separation of crura

Treatment

Symptomatic (Type 2–4) → Surgery
Asymptomatic → Conservative

Complications

  • Gastric volvulus
  • Obstruction
  • Ischemia
  • Respiratory failure

Differential Diagnosis

  • Retrocardiac lung abscess
  • Retrocardiac empyema
  • Esophageal diverticulum
